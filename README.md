# TFG - Plataforma de Streaming Musical 🎵

Este proyecto constituye nuestro **Trabajo de Fin de Grado (TFG)**. Se trata de una plataforma web de streaming diseñada para la gestión y visualización de contenido multimedia, desarrollada con una arquitectura moderna y escalable.

## 🚀 Características Principales

* **Streaming de Audio:** Reproducción de alta fidelidad con controles avanzados de audio (play/pause, salto de pista, volumen...).
* **Gestión de Catálogo Musical:** Panel para la organización de canciones, álbumes, géneros y artistas.
* **Playlists Personalizadas:** Sistema de creación y gestión de listas de reproducción por parte de los usuarios.
* **Arquitectura REST:** API escalable construida con **Spring Boot** para el manejo de metadatos y archivos.
* **Interfaz Adaptativa:** Experiencia de usuario fluida y moderna desarrollada en **Angular** con diseño responsive.
* **Persistencia de Datos:** Almacenamiento eficiente de perfiles de usuario y bibliotecas mediante **MySQL**.
